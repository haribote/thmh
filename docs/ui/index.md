# ui

The human-facing catalog: pages, layouts, and components. Feature IDs in this domain are split by kind:

- `UIC` — components
- `UIL` — layouts
- `UIP` — pages
- `UIX` — other (features that do not appear directly on screen, e.g. the catalog BFF connection)
