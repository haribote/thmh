# mcp

The agent-facing MCP server and its tools. Feature IDs in this domain use the `MCP` prefix.

| ID | Title | Summary |
| --- | --- | --- |
| [MCP001](MCP001_mcp-server.md) | MCP server | One tool surface over two transports, reading a catalog source chosen independently of either. |
| [MCP002](MCP002_search-components.md) | search_components | Finds candidates by matching terms across four fields, and says where each landed. |
