# cli

The `thmh` command-line interface. Feature IDs in this domain use the `CLI` prefix.

| ID | Title | Summary |
| --- | --- | --- |
| [CLI001](CLI001_build-command.md) | build command | One analysis pass written to `catalog.json`, with no dev server. |
| [CLI002](CLI002_init-command.md) | init command | Adds the plugin to an existing Vite config, and refuses when the shape is not recognized. |
| [CLI003](CLI003_dev-command.md) | dev command | Serves the catalog for a repository with no host app, resolving Vite from the target. |

See [requirements](../requirements.md) and [CONTRIBUTING.md](../../CONTRIBUTING.md).
